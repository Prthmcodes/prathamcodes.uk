# PRTHM_CODES — Portfolio v2.0

My personal portfolio — a single-page, cyberpunk-styled site with a real-time **Three.js/WebGL 3D scene** (wireframe icosahedron, particle galaxy, infinite grid floor), glitch typography, a terminal-style about section, custom cursor, and HUD elements.

🌐 **Live:** [https://prathamcodes.uk](https://prathamcodes.uk)

## Built with

- Vanilla HTML / CSS / JavaScript — single self-contained `index.html`
- [Three.js](https://threejs.org) (r128) for the 3D background
- Google Fonts: Unbounded, JetBrains Mono, Space Grotesk
- Hosted on **GitHub Pages** with a custom domain

## Featured projects on the site

- [Airola](https://github.com/Prthmcodes/airola) — full-stack turnover-cleaning platform (React 19, TypeScript, Supabase) — live at [airola.co.uk](https://airola.co.uk)
- [PromptLab AI](https://github.com/Prthmcodes/promptlab-ai) — AI prompt optimizer (Next.js 16, TypeScript) — live at [promptlab-ai-ruby.vercel.app](https://promptlab-ai-ruby.vercel.app)
